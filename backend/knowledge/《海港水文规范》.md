## 2. 航道尺度计算规范 (6.4节)

*   **单线航道**: $W = A + 2c$

### 2.2 航道水深计算 (6.4.6)
*   **通航水深 ($D_0$)**: $D_0 = T + Z_0 + Z_1 + Z_2 + Z_3$
*   **设计水深 ($D$)**: $D = D_0 + Z_4$
*   **参数含义**:
    *   $Z_0$: 航行下沉量
    *   $Z_1$: 龙骨下最小富裕深度 (见表6.4.6-1)
    *   $Z_2$: 波浪富裕深度 (见表6.4.6-2)
    *   $Z_3$: 纵倾富裕深度 (油船/散货船取0.15m)
    *   $Z_4$: 备淤深度 (不小于0.4m)

## 3. 跨越航道建筑物 (6.3节)

### 3.1 通航净空高度 (6.3.1)
*   **公式**: 净空高度 = 代表船型水线以上高度($H_{air}$) + 富裕高度($H_{safe}$)
*   **$H_{safe}$ 取值**:
    *   内河/有掩护海域: **2.0m**
    *   开敞海域重要航道: **4.0m**

### 3.2 航道转弯段 (6.3.7)
*   **10° < φ ≤ 30°**: $R = (3 \sim 5)L$，切角法加宽。
*   **30° < φ ≤ 60°**: $R = (5 \sim 10)L$，折线切割法。
*   **φ > 60°**: $R > 10L$，需模拟试验验证。
