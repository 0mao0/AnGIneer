# 计算设计底高程
Description: 计算航道的设计底高程，包括备淤深度。

## 实施步骤
### Step 1.明确引用规范内容

*   **设计船型尺度**: 请参考 [《海港水文规范》](../knowledge/《海港水文规范》.md) 第 1 章。
*   **计算公式与备淤深度**: 请参考 [《海港水文规范》](../knowledge/《海港水文规范》.md) 第 2.2 节。

### Step 2.核心参数解析与确定

1.  **`T` - 设计船型满载吃水**
    *   参考《海港水文规范》第 1 章船型尺度表。

2.  **`Z₀` ~ `Z₃`**:
    *   同“航道通航底高程”计算步骤，参考《海港水文规范》第 2.2 节。

3.  **`Z₄` - 备淤深度**
    *   **首选路径**: 查阅【已知条件】是否直接提供`Z₄`值。
    *   **备用路径**: 若未提供，参考《海港水文规范》第 2.2 节：对于有淤积的港口，计算结果不宜小于 **0.4m**。

4.  **`H_nav` - 航道设计通航水位**

### Step 3.分步计算

1.  **计算航道通航水深 `D₀`**:
    *   **公式**: `D₀ = T + Z₀ + Z₁ + Z₂ + Z₃`

2.  **计算航道设计水深 `D`**:
    *   **公式**: `D = D₀ + Z₄`

3.  **计算航道设计底标高 `Z`**:
    *   **公式**: `Z = H_nav - D`

### Step 4.内容输出

请通过表格形式输出：通航底标高 `Z`、航道通航水深 `D₀`、航道设计水深 `D`、`T`、`Z₀`、`Z₁`、`Z₂`、`Z₃`、`Z₄`、`H_nav` 等参数的计算结果。
