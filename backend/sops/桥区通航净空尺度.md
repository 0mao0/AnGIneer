# 计算跨越航道建筑物（如桥梁、缆线）的通航净空高度和净空宽度。
Description: Calculate the clearance height and width of a bridge or cableway over a channel.

## 实施步骤
### Step 1.明确引用规范内容

关于通航净空高度 ($H_{air}$, $H_{safe}$) 和净空宽度 ($K$) 的取值标准，请参考 [《海港水文规范》](../knowledge/《海港水文规范》.md) 第 3.1 节和 2.1 节。

### Step 2.核心参数解析与确定

1.  **`H_air` - 代表船型水线以上高度**
    *   **获取路径**: 查阅【已知条件】。这是船舶自身的物理属性（空载最高点 - 吃水）。
    *   **异常处理**: 若缺失，需提示用户：“请提供代表船型的水线以上高度(Air Draft)”。

2.  **`H_safe` - 垂直富裕高度**
    *   **判定逻辑**: 检查【已知条件】中的水域环境描述。
    *   参考《海港水文规范》第 3.1 节：
        *   若提及“开敞海域”、“波浪大”且“重要航道” $\rightarrow$ 取 **4.0m**。
        *   其他情况（内河、港内、有掩护） $\rightarrow$ 默认取 **2.0m**。

3.  **`W` - 航道通航宽度**
    *   **获取路径**: **继承**上一环节【通航宽度】的计算结果。

4.  **`K` - 净空宽度扩大系数**
    *   **交互逻辑**:
        1.  **首选**: 用户直接输入 $K$ 值。
        2.  **推荐**: 若用户未输入，Agent需分析上下文：
            *   若 **DWT $\ge$ 10万吨** 或 **环境=开敞/风浪大** $\rightarrow$ 推荐 $K = 1.8$。
            *   若 **环境=掩护良好** $\rightarrow$ 推荐 $K = 1.5$。
            *   若介于两者之间 $\rightarrow$ 推荐 $K = 1.6 \sim 1.7$。

### Step 3.最终结果计算

1.  **计算净空高度 ($H_{clearance}$)**
    *   **公式**: $H_{clearance} = H_{air} + H_{safe} + \text{修正项}$

2.  **计算净空宽度 ($B$)**
    *   **公式**: $B = K \times W$

### Step 4.成果输出

列出净空高度、净空宽度、`H_air`、`H_safe`、`W`、`K` 等参数。
