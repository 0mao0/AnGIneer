# 更新日志 (CHANGELOG)

所有对 PicoAgent 项目的显著更改都将记录在此文件中。

## [0.0.2] - 2026-01-25

### 🚀 新增 (Added)
- **前端模型选择**: 在测试面板中增加了模型选择下拉框，默认设置为 Qwen-VL。
- **推理时间显示**: 意图路由结果中新增推理耗时（Duration）显示。
- **流程图动态效果**: 运行中的流程图节点会自动居中并保持动态向下的视觉感。

### 🔧 优化 (Changed)
- **流程图稳定性**: 修复了 Mermaid 渲染中的括号解析错误，确保节点标签在包含特殊字符时能正确显示。
- **数据同步**: 修复了前端无法正确显示后端路由原因（Reason）的问题，完善了 `api_bridge.py` 与前端的数据对接。
- **GitHub 兼容性**: 修复了 `README.md` 中的 Mermaid 语法，使其在 GitHub 上能正常渲染。
- **忽略项更新**: 在 `.gitignore` 中增加了 `.trae/` 文件夹。

## [0.0.1] - 2026-01-18

### 🚀 新增 (Added)
- **混合架构基础**: 实现了规则执行与 LLM 决策相结合的混合 Agent 调度引擎。
- **SOP 智能加载器**: 支持 Markdown 格式 SOP 的自动解析与 LLM 预分析，提取输入参数与注意事项。
- **NVIDIA API 集成**: 默认支持 NVIDIA 提供的多种大模型（如 Nemotron-3-nano, DeepSeek-v3.2, Kimi 等）。
- **工具集**: 
  - 通用工具：计算器、智能查表 (Table Lookup)、知识检索 (RAG)。
  - GIS工具初试试：实现断面工程量体积计算。
- **双语支持**: 工具描述支持中英双语，代码注释与日志全面本地化。

### 🔧 优化 (Changed)
- **重构路由层**: 将 `router.py` 重命名为 `classifier.py`，优化意图识别逻辑与加载顺序。
- **项目文档**: 全面重写 README.md，包含架构图、路线图及快速开始指南。
